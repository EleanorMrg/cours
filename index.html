Définitions

Une page web est une page écrite en HTML et CSS (et JS) ; elle est interprétée par le navigateur (chrome, firefox,internet-explorer,safari).

La balise "i" permet de mettre en italique un texte

La balise "u" permet de souligné un texte

les différents types de balises

Il y a deux types de balise :

  • les balises usuelles qui encadrent du code HTML. Le code encadré fini par </balise > ;
    • ᘹ &
  • les balises orphelines qui n'encadrent aucun code et qui se ferment elles-mêmes ; par exemple :
    < br/ >
    < hr/ >
    < img/ >
    < input/ >

les balises peuvent avoir des attributs

un attribut désigne une caractéristique d'une balise. Cette balise est interprété comme objet par le navigateur, elle prend une valeur par défault à l'initialisation du site mais elle peut être modifiée.
Exemple <p style="color:red">
